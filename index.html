Image Quality Testing Software

Developed a sophisticated Image Quality Testing Software for ADAS Cameras and Camera Monitoring Systems (CMS) in the Automotive Industry.

Key Achievements:

  • Developed complete signal processing pipeline including edge profile computation
  • Implemented advanced edge-fitting algorithms using 5th order polynomial modeling

  • Integrated frequency-domain corrections
  • Built interactive Qt-based visualization system
