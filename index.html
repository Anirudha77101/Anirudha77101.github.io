Research Interests

My research focuses on real-time control systems for collaborative robots, perception algorithms for autonomous manipulation, and trajectory optimization for dynamic systems. Current work involves developing low-latency control architectures and robust perception pipelines for industrial applications.

Selected Projects

Implementation of Collaborative Robot Control Architecture

Development of real-time control systems for industrial collaborative robots focusing on latency optimization and human-robot interaction. Achieved significant performance improvements through advanced middleware integration.

  • Reduced control loop latency from 20ms to 100ns through optimized ROS2-DDS bridge
  • Implemented deterministic QoS policies for real-time control requirements
  • Developed intuitive human-robot interaction interface
ROS2 RTI Connext DDS Real-time Systems

Perception Algorithms for Dynamic Manipulation

Research on robust perception algorithms for autonomous manipulation in dynamic environments. Developed integrated system achieving top rankings in international competition.

  • Implemented hybrid YOLO-V4 and ICP pipeline for 6-DOF pose estimation
  • Achieved 85% success rate in dynamic object manipulation tasks
  • Developed task and motion planning integration for long-horizon tasks
Computer Vision Deep Learning Motion Planning

MTF-based Image Quality Analysis System

Implementation of ISO 12233 standard for quantitative image quality assessment through Modulation Transfer Function analysis.

  • Developed complete signal processing pipeline for edge-based MTF analysis
  • Implemented real-time processing capabilities for live analysis
  • Created visualization tools for results interpretation
Signal Processing C++ Computer Vision

Vehicle Dynamics Control Systems

Research on advanced control systems for electric vehicles including torque vectoring and traction control optimization.

  • Developed gain-scheduled PID control system for torque vectoring
  • Implemented fuzzy logic-based traction control
  • Achieved 0.3s improvement in cornering performance
